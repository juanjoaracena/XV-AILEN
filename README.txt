# Invitación XV Ailén (proyecto listo)

Este proyecto es una mini web (un solo archivo HTML) con:
- Sobre animado (abrir invitación)
- Tarjeta con detalles
- Botón de WhatsApp para confirmar
- Botón de Google Maps
- Botones para copiar Alias/CVU
- Música opcional (si agregás un archivo `audio.mp3`)

## Archivos
- `index.html` (todo está acá)

## Cómo usarlo (rápido)
1) Descomprimí el ZIP.
2) Abrí `index.html` con Chrome/Edge y probalo.
3) (Opcional) Poné un archivo llamado **audio.mp3** en la misma carpeta.

> Nota por derechos de autor: no incluimos audio comercial dentro del proyecto.
> Si querés música, subí tu propio MP3 (por ejemplo un audio que tengas con permiso o una versión libre).

## Cómo ponerlo en Google Sites (recomendado)
Google Sites funciona mejor si embebés **una URL** (una página publicada).

Opción A (muy simple): GitHub Pages
1) Creá un repo en GitHub (por ejemplo: `xv-ailen`).
2) Subí `index.html` (y `audio.mp3` si lo usás).
3) Activá Pages: Settings → Pages → Deploy from a branch → main / root.
4) Te va a dar una URL tipo `https://TUUSUARIO.github.io/xv-ailen/`
5) En Google Sites: Insertar → **Incorporar** → URL → pegás la URL.

Opción B: Netlify (drag & drop)
1) Entrá a Netlify, arrastrá la carpeta con `index.html`.
2) Te entrega una URL.
3) En Google Sites: Insertar → Incorporar → URL.

## Editar datos (en index.html)
Buscá esta parte y cambiá lo que haga falta:
- WHATSAPP_NUMBER
- WHATSAPP_MSG
- MAPS_URL
